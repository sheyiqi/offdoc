#### Read Assistance

[Company Name] memory compilers provide the Read Assistance feature.

The `RA` pin provides control settings for Word Line Under-Drive to improve the SRAM read capability. The read capability of a memory instance can be adjusted through the external input pin `RA[1:0]`. The read capability increases as `RA[1:0]` changes sequentially from `00`, `01`, `10`, to `11`.
