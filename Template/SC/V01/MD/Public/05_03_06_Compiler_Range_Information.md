The Column Mux is an option which provides different physical aspect ratio of the instance generated by the compiler. The larger Column Mux value correspond the higher selection levels of instance I/O.

If the instance has a high number of words and a low number of bits per words, use the highest possible Column Mux value to obtain a square aspect ratio. If the instance has a high number of bits per word and a low number of words, use the lowest possible Column Mux value.
