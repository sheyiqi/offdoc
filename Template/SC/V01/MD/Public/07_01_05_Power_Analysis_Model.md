#### Power Analysis Model

[Company Name] memory compiler supports the power analysis model `Apache Virtual Model (AVM)`.

##### Using AVM

The AVM view `(*.avmcfg)` describes electrical properties of a memory instance and is used by the `avm utility` of `Redhawk` tool to generate current and capacitance profile.

Use the AVM view to perform power analysis with the help of `Redhawk` tool from Ansys.
