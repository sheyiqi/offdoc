#### ECC

[Company Name] memory compilers provide ECC to correct soft errors.

When the `ECC feature` is enabled, each word includes additional bits to store ECC information. The memory instance is also wrapped in a synthesized RTL wrapper to perform `SECDED` on each word.

If the compiler supports the `Bit Write feature`, `ECC` and `Bit Write` cannot be configured in the same instance. Otherwise, incorrect error detection and correction may occur.
