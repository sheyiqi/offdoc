#### Periphery Off

[Company Name] memory compilers provide the Periphery Power Supply Off feature.

When the `Dual Rail feature` and the `Power Gating feature` are selected, the `Periphery Off feature` can be enabled. The instance generates the `POFF` and `POFFTUNE` pins, both of which are in the VDDC domain.

The periphery off mode allows the external VDD and VDDC power supplies to be powered off independently. This feature takes effect only when both the `Dual Rail feature` and the `Power Gating feature` are enabled.
