#### CEN-Gating

[Company Name] memory compilers optionally support the Chip Enable Gating feature.

By default, the address `AA/AB` pins and data input `DA` pins are not gated by the `CENA/CENB` pin may increase toggle power. In this case, `CENA/CENB` requires a relatively small setup time.

To reduce the toggle power of the address `AA/AB` pins and data input `DA` pins, users can enable the `CEN-Gating feature`; however, the setup time of `CENA/CENB` may increase.
