#### Bist Mux

[Company Name] memory compilers provide built-in BIST circuit functionality.

The BIST Multiplexer is a crucial front-end interface module responsible for controllable switching of SRAM addresses, input data, chip select, control signals, and output data between function mode and test mode. The `BIST Multiplexer` generates enable signals for switching between the two modes.

- Function mode: When the BIST Multiplexer fails, the SRAM accepts address, data, and control signals from the chip's main logic.
- Test mode: When the BIST Multiplexer is enabled, the SRAM accepts address, data, and control signals generated by the BIST Controller, and executes the MBIST algorithm (all 0s, all 1s, checkerboard pattern, etc.).
