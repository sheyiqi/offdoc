### Expanded Instance Naming Rules

The automatic memory name of an instance is generated by combining the information from the compiler name itself with the values of certain instance parameters entered for the component. For example, the default naming calculation will generate an instance name in the following format:`SCT7HHS2PRF1024X32M4B1W1C1T0P0R1`The breakdown of the name is as follows, see Table 6-1 for details:`SC T7 H HS2PRF 1024 X 32 M4 B1 W1 C1 T0 P0 R1`
